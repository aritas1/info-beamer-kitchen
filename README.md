# Info-Beamer for your kitchen
===================

## Depends on:
* https://github.com/aamuuninen/info-beamer
* https://github.com/aamuuninen/info-beamer-nodes
* https://github.com/aritas1/barcodescanner

## Requirements:
* Java 7
* Lua 5.1
* C
* Linux System
* Sudo
* Udev Servie


## Installing

For install run:
`./install.sh`

## Updating

For update run:
`./update.sh`


## Executing
`./run.sh`
